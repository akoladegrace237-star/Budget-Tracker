# 💰 Budget Tracker - Web App

A comprehensive budget management web application that helps you track income, expenses, savings, investments, and plan for the future. Access it anytime from your phone or desktop!

## 🌟 Features

✨ **Complete Budget Management**
- Monthly income tracking (salary, bonuses, side income)
- Detailed expense categorization (housing, utilities, food, transportation, healthcare, entertainment)
- Savings and investment monitoring
- Recurring expense tracking (credit card payments, loans, subscriptions)
- 12-month financial projections

📊 **Visual Analytics**
- Interactive dashboard with summary cards
- Expense breakdown pie charts
- Income vs. expenses comparisons
- Net savings trend analysis
- Monthly projections

💰 **Smart Features**
- Automatic calculations and formulas
- Data saved locally in your browser
- Export to Excel for backup
- Mobile-responsive design
- Works completely offline after first load

📱 **Access Anywhere**
- Works on phones, tablets, and desktops
- No app installation needed
- Accessible via GitHub Pages link anytime

## 🚀 Quick Start

### Option 1: Access Web App (Easiest)
1. Visit your deployed GitHub Pages link
2. Start entering your budget data
3. Data is saved automatically in your browser
4. Use "Export to Excel" to backup anytime

### Option 2: Run Locally
1. Clone the repository
2. Open `index.html` in your web browser
3. Start managing your budget

## 🌐 Deploy to GitHub Pages (Access from Your Phone!)

### Step 1: Push to GitHub
```bash
git add .
git commit -m "Add budget tracker web app"
git push origin main
```

### Step 2: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under "Source", select **main** branch
4. Click **Save**
5. GitHub will provide you with your live URL

### Step 3: Access from Your Phone
1. Copy the provided GitHub Pages URL
2. Open it in your phone's browser
3. Bookmark it for quick access
4. Your budget app is now always in your pocket!

**Example URL:** `https://yourusername.github.io/Bugdets-App/`

## 💡 Tips for Success

1. **Update monthly**: Enter your actual figures soon after the month ends
2. **Track recurring**: Add all recurring expenses and loans
3. **Review trends**: Check charts to identify spending patterns
4. **Plan ahead**: Use projections to set realistic financial goals

## 🔒 Data Privacy

- **All data is stored locally** in your browser using LocalStorage
- **No data is sent to any server**
- Your budget information stays on your device
- Use "Export to Excel" to backup your data
